Este é um projeto de um site de cursos desenvolvido com o objetivo de consolidar fundamentos de desenvolvimento web.
O foco principal aqui não é apenas o resultado final, mas o processo de aprendizado contínuo e a superação de desafios técnicos.

O propósito deste repositório é servir como um laboratório de prática. Através dele, busco identificar erros de lógica,
design e estruturação, aplicando correções e melhorias conforme evoluo nos meus estudos.

Obrigado!
